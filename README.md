# Secret Number Game

This project is a simple guessing game developed to practice JavaScript fundamentals, DOM manipulation, and basic logic.

The goal of the game is to guess the secret number generated randomly by the system.

## How It Works
- The system generates a random number within a defined range.
- The player enters a number and receives feedback indicating whether the secret number is higher or lower.
- The game tracks the number of attempts until the correct answer is found.
- Once the player wins, the restart button becomes available.

## Features
- Random number generation without repetition
- Attempt counter
- User feedback through text and voice
- Game reset functionality

## Technologies Used
- HTML
- CSS
- JavaScript

## Purpose
This project was created for learning purposes to practice JavaScript logic, functions, conditionals, loops, and interaction with the DOM.
